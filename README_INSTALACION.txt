GETINWEB - VERSION PHP DESDE TU PROYECTO ORIGINAL

Esta versión conserva tu index original y tu estilos.css original.
No se reemplazó el diseño del inicio.

PASOS EN XAMPP:

1. Copiá la carpeta GetInWeb_PHP_DESDE_TU_PROYECTO dentro de:
   C:\xampp\htdocs\

2. Abrí XAMPP y activá:
   - Apache
   - MySQL

3. Entrá en el navegador a:
   http://localhost/phpmyadmin

4. Importá el archivo:
   getinweb.sql

5. Abrí la página en:
   http://localhost/GetInWeb_PHP_DESDE_TU_PROYECTO/index.php

IMPORTANTE:
- index.php usa estilos.css, que es tu CSS original.
- Las páginas nuevas usan php_extra.css.
- Si no estás logueado y tocás Crear ahora / Explorar Plantillas, va a login.php.
- Si estás logueado, va a explorador.php.
